# 🏔️ Панорама карьера + Детекция трещин

Веб-приложение для сшивки панорам и анализа трещин на горных породах с помощью нейросети YOLO.

## 🚀 Возможности

- Загрузка 5–20 фото (JPEG, PNG, TIF)
- Автоматическая сшивка панорамы
- Поиск трещин с помощью YOLO (модель `best.pt`)
- Анализ тепловизионных снимков (горячие зоны)
- Современный интерфейс в стиле ГОК "Томинский"
- Скачивание результатов (панорама + размеченное изображение)

## 🛠️ Как установить и запустить

1. Клонируй репозиторий:
   ```bash
   git clone https://github.com/nikitosrus01/BC-26.git
   cd BC-26/bv   # если проект в папке bv
2. Создай виртуально окружение
    python -m venv .venv
    .venv\Scripts\activate   
3. Устнови библиотеки
    pip install flask opencv-python ultralytics numpy Pillow
4. Запусти
    python app.py
5. Открой в браузере 
    http://127.0.0.1:5000

🖼️ Как пользоваться
    Выбери режим: Трещины (обычные фото) или Тепловизор.
    Перетащи папку с фотографиями или выбери файлы.
    Нажми «Запустить обработку».
    Через 1-3 минуты скачай панораму и результат.
